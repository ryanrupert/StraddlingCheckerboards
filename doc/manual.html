Straddling Checkerboards

repository - https://github.com/ryanrupert/StraddlingCheckerboards

Installing

To install Straddling Checkerboards take the files listed below and move them into your project:

  • Encoding.h
  • Maps.cpp
  • Encode.cpp
  • Decode.cpp

Starting

To start create an object and name it something. 

            
                Crypto obj_name;
Then you need to set the language for the program. 
            
                obj_name.setLang("EN");
Straddling Checkerboard supports two letter language codes. The languages that it supports can be found here To encode a string you would use the code below. (In this document I will be using CT1 as the table. More Info). 
            
                encoded_text = obj_name.Encode("CT1", "THIS IS A TEST STRING");
NOTE: Straddling checkerboard is currently case sensitive. And will only except capital letters.


To decode a string you would use the code below. 
            
                plain_text = obj_name.Encode("CT1", "62836");
NOTE: Spaces are not allowed in the text going in for decoding. And the string going in for decoding may only be numbers.

Tables

The table is simply the list of letters that are paired to a number. You can use any table in the table.xml document as long as the language you are using supports that table.

Functions

Here is a list of the public members for this object.
Member Purpose
encode This member encodes text
decode This member will decode cipher text
setLang This member sets the language of the object
getLang This member will get the objects current language

encode

            
                std::string Crypto::encode(std::string tableid, std::string plain);

Parameters

tableid - String with the name of the table to use for encoding.
plain - String with the plain text to be encoded

Return Value

String that contains the encoded text

decode

            
                std::string Crypto::decode(std::string tableid, std::string encoded);

Parameters

tableid - String with the name of the table to use for encoding.
encoded - String with the cipher text to be decoded

Return Value

String of the plain text

setLang

            
                void Crypto::setLang(const char *language);

Parameters

language - Char with the language code to set the object to.

Return Value

none

getLang

            
                char* Crypto::getLang();

Parameters

none

Return Value

Returns char pointer to the current language
