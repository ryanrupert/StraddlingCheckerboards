Straddling Checkerboards

repository - https://github.com/ryanrupert/StraddlingCheckerboards

Installing

To install Straddling Checkerboards take the files listed below and move them into your project:

  • Encoding.h
  • Maps.cpp
  • Encode.cpp
  • Decode.cpp

Starting

To start create an object and name it something. 

            
                Crypto obj_name;
Then you need to set the language for the program. 
            
                obj_name.setLang("EN");
Straddling Checkerboard supports two letter language codes. The languages that it supports can be found here To encode a string you would use the code below. (In this document the table I will be using is CT1 for the table. More Info). 
            
                encoded_text = obj_name.Encode("CT1", "THIS IS A TEST STRING");
Note Straddling checkerboard is currently case sensitive. And will only except capital letters. To decode a string you would use the code below. 
            
                plain_text = obj_name.Encode("CT1", "62836");
Note spaces are not allowed in the text going in for decoding. And the string going in for decoding may only be numbers.

Tables

The table is simply the list of letters that are paired to a number. You can use any table in the table.xml document as long as the language you are using supports that table.
